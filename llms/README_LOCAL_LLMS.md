# Install Local LLMs

## Ollama 

### Download Ollama

Go To the Web site: https://ollama.com/download
Download for Mac, Windows and Linux.

For Linux
```
curl -fsSL https://ollama.com/install.sh | sh
```

### Meta (FaceBook) Llama3
```
ollama run llama3
```

### Mistral

```
ollama run mistral
```

### Microsoft PHI-3
```
ollama run phi3
```

### Google Gemma
```
ollama run gemma
```
