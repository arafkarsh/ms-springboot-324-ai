# Install Local LLMs

## Ollama 

### Download Ollama

Go To the Web site: https://ollama.com/download
Download for Mac, Windows and Linux.

For Linux
```
curl -fsSL https://ollama.com/install.sh | sh
```

### 1. Meta (FaceBook) Llama3
```
ollama run llama3
```

### 2.Mistral

```
ollama run mistral
```

### 3. Microsoft PHI-3
```
ollama run phi3
```

### 4. Google Gemma
```
ollama run gemma
```
